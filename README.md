# 提示词生成器

## 项目简介
这是一个基于PyQt5开发的提示词生成工具，可以帮助用户快速创建和管理各种提示词模板。

## 功能特点
- 技能模板管理：添加、编辑、删除技能模板
- 内容替换：使用{content}占位符动态替换内容
- 实时预览：编辑时实时查看生成结果
- 一键复制：快速复制生成内容到剪贴板

## 安装指南
1. 确保已安装Python 3.6+
2. 安装依赖：`pip install PyQt5`
3. 运行程序：`python main_window.py`

## 使用方法
1. 添加技能模板
2. 选择模板并填写内容
3. 预览并复制生成结果
